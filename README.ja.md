# 韓国で暮らすための実用リソース集

> ビザ、住まい、健康保険、税金、仕事、暮らしのルールをまとめました。

韓国に住む外国人のために [Seoulstart](https://seoulstart.com) が管理するキュレーションリストです。公式ポータルと一次情報を先頭に置いています。

> **メモ**: この日本語版は英語版READMEの要約です。Seoulstartの全ガイドは英語版サイト [seoulstart.com](https://seoulstart.com) でご覧いただけます。

## 主要な公式ポータル

- [HiKorea](https://www.hikorea.go.kr/)
- [Government 24 (정부24)](https://www.gov.kr/)
- [Hometax (홈택스)](https://www.hometax.go.kr/)
- [NHIS (국민건강보험)](https://www.nhis.or.kr/)
- [NPS (국민연금)](https://www.nps.or.kr/)
- [Danuri (다누리)](https://www.liveinkorea.kr/)

## トピック別

完全なリストは [英語版README](README.md) にトピック別で整理されています：到着後90日、ビザ、住まい、お金と税金、医療と保険、家族、仕事、韓国語とKIIP、文化、ペット、コミュニティとアプリ、韓国を離れるとき。
