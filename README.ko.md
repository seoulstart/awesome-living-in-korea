# 한국 거주 외국인을 위한 실용 자료 모음

> 한국에서 살아가는 외국인 거주자를 위한 정부 포털, 신뢰할 수 있는 가이드, 계산기, 디렉토리 모음. 비자, 주거, 보험, 세금처럼 설명서가 따라오지 않는 시스템들을 다뤄요.

한국에 거주하는 약 300만 명의 외국인을 위해 [Seoulstart](https://seoulstart.com/ko)가 관리하는 큐레이션 자료 목록이에요.

> **참고**: 본 한국어판은 현재 영문 README의 요약 안내본이에요. 전체 번역은 진행 중이고, 그동안 Seoulstart의 한국어 가이드는 [seoulstart.com/ko](https://seoulstart.com/ko)에서 바로 보실 수 있어요.

## 주요 정부 포털

- [하이코리아 (HiKorea)](https://www.hikorea.go.kr/) - 법무부 출입국 포털. 비자, 외국인등록증, e-Form 신청.
- [정부24](https://www.gov.kr/) - 대부분의 정부 서비스를 하나의 로그인으로.
- [홈택스](https://www.hometax.go.kr/) - 국세청 온라인 신고 포털.
- [국민건강보험공단 (NHIS)](https://www.nhis.or.kr/) - 건강보험 가입, 보험료, 본인부담.
- [국민연금공단 (NPS)](https://www.nps.or.kr/) - 국민연금 가입, 환급.
- [다누리](https://www.liveinkorea.kr/) - 다문화가족 지원 포털, 13개 언어 지원.
