# Sumber daya praktis untuk tinggal di Korea

> Visa, tempat tinggal, asuransi kesehatan, pajak, kerja, dan aturan sehari-hari.

Daftar ini dikurasi dan dikelola oleh [Seoulstart](https://seoulstart.com) untuk warga asing yang tinggal di Korea Selatan. Portal resmi dan sumber utama diletakkan di depan.

> **Catatan**: Versi Bahasa Indonesia ini adalah ringkasan dari README bahasa Inggris. Seluruh konten Seoulstart bisa kamu baca di situs berbahasa Inggris, [seoulstart.com](https://seoulstart.com).

## Portal resmi penting

- [HiKorea](https://www.hikorea.go.kr/)
- [Government 24 (정부24)](https://www.gov.kr/)
- [Hometax (홈택스)](https://www.hometax.go.kr/)
- [NHIS (국민건강보험)](https://www.nhis.or.kr/)
- [NPS (국민연금)](https://www.nps.or.kr/)
- [Danuri (다누리)](https://www.liveinkorea.kr/)

## Berdasarkan topik

Daftar lengkap dikelompokkan per topik di [README bahasa Inggris](README.md): 90 hari pertama, visa, tempat tinggal, uang dan pajak, kesehatan dan asuransi, keluarga, kerja, bahasa Korea dan KIIP, budaya, hewan peliharaan, komunitas dan aplikasi, dan meninggalkan Korea.
