# ကိုရီးယားတွင် နေထိုင်ရန် အသုံးဝင်သော အရင်းအမြစ်များ

> ဗီဇာ၊ နေထိုင်ရေး၊ ကျန်းမာရေးအာမခံ၊ အခွန်၊ အလုပ်နှင့် နေ့စဉ်ဘဝစည်းမျဉ်းများ။

ဤစာရင်းကို ကိုရီးယားတွင် နေထိုင်သော နိုင်ငံခြားသားများအတွက် [Seoulstart](https://seoulstart.com) မှ စုစည်းထိန်းသိမ်းထားသည်။ တရားဝင်ပေါ်တယ်များနှင့် မူရင်းအရင်းအမြစ်များကို ရှေ့ဆုံးတွင် ထားရှိသည်။

> **မှတ်ချက်**: ဤမြန်မာဘာသာ ဗားရှင်းသည် အင်္ဂလိပ် README ၏ အကျဉ်းချုပ်ဖြစ်သည်။ Seoulstart ၏ အကြောင်းအရာအပြည့်အစုံကို အင်္ဂလိပ်ဆိုက် [seoulstart.com](https://seoulstart.com) တွင် ကြည့်နိုင်သည်။

## အဓိက တရားဝင်ပေါ်တယ်များ

- [HiKorea](https://www.hikorea.go.kr/)
- [Government 24 (정부24)](https://www.gov.kr/)
- [Hometax (홈택스)](https://www.hometax.go.kr/)
- [NHIS (국민건강보험)](https://www.nhis.or.kr/)
- [NPS (국민연금)](https://www.nps.or.kr/)
- [Danuri (다누리)](https://www.liveinkorea.kr/)

## ခေါင်းစဉ်အလိုက်

စာရင်းအပြည့်အစုံကို [အင်္ဂလိပ် README](README.md) တွင် ခေါင်းစဉ်အလိုက် စီစဉ်ထားသည်- ပထမ ၉၀ ရက်၊ ဗီဇာ၊ နေထိုင်ရေး၊ ငွေနှင့်အခွန်၊ ကျန်းမာရေးနှင့်အာမခံ၊ မိသားစု၊ အလုပ်၊ ကိုရီးယားဘာသာနှင့် KIIP၊ ယဉ်ကျေးမှု၊ အိမ်မွေးတိရစ္ဆာန်၊ အသိုက်အဝန်းနှင့်အက်ပ်များ၊ ကိုရီးယားမှ ထွက်ခွာခြင်း။
