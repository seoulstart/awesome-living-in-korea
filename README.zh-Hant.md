# 在韓國生活的實用資源整理

> 簽證、住房、健康保險、稅金、工作和生活規則。

這份清單由 [Seoulstart](https://seoulstart.com) 為在韓國生活的外國人整理與維護。官方入口網站和第一手來源排在前面。

> **說明**: 這份繁體中文版是英文 README 的摘要。Seoulstart 的完整內容在英文網站 [seoulstart.com](https://seoulstart.com) 上。

## 主要官方入口網站

- [HiKorea](https://www.hikorea.go.kr/)
- [Government 24 (정부24)](https://www.gov.kr/)
- [Hometax (홈택스)](https://www.hometax.go.kr/)
- [NHIS (국민건강보험)](https://www.nhis.or.kr/)
- [NPS (국민연금)](https://www.nps.or.kr/)
- [Danuri (다누리)](https://www.liveinkorea.kr/)

## 按主題

完整清單在 [英文 README](README.md) 裡按主題分類：抵達後90天、簽證、住房、金錢與稅金、醫療與保險、家庭、工作、韓語與KIIP、文化、寵物、社群與App、離開韓國。
