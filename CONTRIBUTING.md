# Contributing

Thanks for helping make this list more useful for foreign residents in Korea.

## What we accept

- Government portals and primary sources (HiKorea, Danuri, NHIS, NTS, gov.kr).
- Free or freemium tools, calculators, and directories that genuinely help.
- Guides and explainers that go past surface-level and link to primary sources.
- Multilingual resources (Vietnamese, Filipino, Russian, Mandarin Chinese, Korean for heritage readers).

## What we don't accept

- Paid newsletters and gated content as the only link for a topic.
- Real estate listings, ad-driven blogs, sponsored placements.
- Resources written for tourists rather than residents.
- Outdated guides that have not been updated for current rules (visa, tax, NHIS).

## How to add a resource

1. Open a pull request that edits `README.md` (or the relevant section file).
2. Add one line in the format `- [Name](URL) - short, factual description of what it covers.`
3. Keep alphabetical or topical order within the section.
4. If your resource is in a language other than English, add the language code in brackets at the end: `[ko]`, `[vi]`, `[fil]`, `[ru]`, `[zh]`.
5. In the PR description, say briefly why this resource is worth adding and how it has helped foreign residents you know.

## Style rules

- Plain language. One sentence per line item, factual not promotional.
- No em dashes. Use commas, colons, or a new sentence.
- "Foreign resident" instead of "expat" as the default term.
- Korean terms: English first, Korean in parentheses on first mention. Example: `community service center (주민센터)`.

## Reporting a broken or outdated link

Open an issue using the "Report broken or outdated resource" template. We try to triage these within a week.

## Code of Conduct

This project follows the [Contributor Covenant](CODE_OF_CONDUCT.md). Be kind.
